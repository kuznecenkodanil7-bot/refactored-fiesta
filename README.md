# Minesweeper Client — Fabric 1.21.5

Клиентский мод для Minecraft Java 1.21.5. По нажатию клавиши `Ж` открывается экран игры «Сапёр».

## Что есть

- Открытие по клавише `Ж` на русской раскладке (`;` на QWERTY).
- Три уровня сложности:
  - Лёгкий: 9×9, 10 мин
  - Средний: 16×16, 40 мин
  - Сложный: 30×16, 99 мин
- ЛКМ открывает клетку.
- ПКМ ставит или снимает флаг.
- Первый клик безопасный: мины не ставятся вокруг первой открытой клетки.
- Кнопка «Заново» перезапускает текущую сложность.

## Сборка

Нужен JDK 21 и Gradle 8.12+.

```bash
gradle build
```

Готовый `.jar` появится в:

```text
build/libs/minesweeper-client-1.0.0.jar
```

## Установка

1. Установи Fabric Loader для Minecraft 1.21.5.
2. Установи Fabric API для Minecraft 1.21.5.
3. Скопируй собранный `minesweeper-client-1.0.0.jar` в папку `.minecraft/mods`.
4. Запусти игру через профиль Fabric 1.21.5.

## Настройка клавиши

По умолчанию используется `GLFW_KEY_SEMICOLON`, то есть клавиша `;` на QWERTY и `Ж` на русской раскладке. В игре её также можно поменять в настройках управления в категории «Сапёр».
